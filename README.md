# Alexa Voice Assistant using Python

## Description

A simple Python-based voice assistant that listens to voice commands and responds using speech. The assistant can play songs on YouTube, search Wikipedia, tell jokes, and provide the current date and time.

## Features

- Speech Recognition
- Text-to-Speech Response
- Play YouTube Songs
- Wikipedia Search
- Current Date and Time
- Joke Generator

## Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- PyWhatKit
- Wikipedia
- PyJokes
- Datetime

## Installation

Install the required libraries:

pip install SpeechRecognition

pip install pyttsx3

pip install pywhatkit

pip install wikipedia

pip install pyjokes

pip install pyaudio

## Run the Project

python alexa.py

## Sample Commands

- Play a song
- What is Artificial Intelligence?
- Tell me a joke
- What time is it?
- Who is Elon Musk?
